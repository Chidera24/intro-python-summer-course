# Intro to Python Summer Course
Into to Python summer course for undergraduate students in the LSAMP program conducting summer research across all STEM fields

## Useful git commands
- `git add filename` - adds a file for staging.
- `git commit -m "A commit message"` - adds staged files for commit and adds a commit message with `-m`.
- `git push` - pushes changes to upstream.
- `git status` - get changes, branch and git info.
- `git config --global user.name "Your Name"` - config your username for your global git.
- `git config --global user.email "youremail@yourdomain.com"` - config your email for your global git.
- `git pull` - fetch changes from upstream.
